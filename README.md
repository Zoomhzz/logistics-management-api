# logistics-management-api
🚀 Logistics &amp; Enterprise Core API | Sistema de gestão corporativa e logística desenvolvido com Java 17 e Spring Boot 3. Implementa o controle integrado de Clientes, Funcionários, Entregadores e Produtos, com foco em integridade de dados (validação de documentos/lotes), persistência via Spring Data JPA e tratamento global de exceções.
